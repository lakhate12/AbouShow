## 📲 Installation

1. Télécharge le fichier `aboushow.apk` sur ton téléphone Android.
2. Ouvre-le. Android demandera l'autorisation d'installer des applications depuis une source inconnue.
3. Active **« Autoriser depuis cette source »** puis appuie sur **Installer**.
4. Lance **AbouShow**. 🎬

> ℹ️ Cet APK est distribué hors du Play Store (sideload). Android peut afficher un avertissement de sécurité standard lors de l'installation d'apps hors-store — c'est normal.

## ⚠️ Compatibilité

- Android **7.0** (minSdk 24) ou plus récent.
